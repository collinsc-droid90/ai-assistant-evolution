# The Inspirer MVP

## What It Does

The Inspirer is a web-based tool designed to help graphic designers overcome creative blocks and jumpstart their ideation process. When you're staring at a blank canvas and need a creative spark, this app generates unique, structured design prompts by randomly combining three key elements:

Design Style - An aesthetic direction (e.g., Minimalism, Pop Art, Brutalism)

Target Audience - A specific demographic to design for (e.g., Gen Z, Small Business Owners, Healthcare Workers)

Emotion/Concept - A feeling or message to convey (e.g., Nostalgia, Innovation, Confidence)
